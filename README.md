# Ex04 Places Around Me
## Date: 08.10.2025

## AIM
To develop a website to display details about the places around my house.

## DESIGN STEPS

### STEP 1
Create a Django admin interface.

### STEP 2
Download your city map from Google.

### STEP 3
Using ```<map>``` tag name the map.

### STEP 4
Create clickable regions in the image using ```<area>``` tag.

### STEP 5
Write HTML programs for all the regions identified.

### STEP 6
Execute the programs and publish them.

## CODE
```
home.html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>WORLD MAP</title>
</head>
<body>
    <!-- Image Map Generated by http://www.image-map.net/ -->
<img src="home.png" usemap="#image-map">

<map name="image-map">
    <area target="" alt="INDIA" title="INDIA" href="india.html" coords="534,335,628,446" shape="rect">
    <area target="" alt="AUSTRALIA" title="AUSTRALIA" href="australia.html" coords="954,734,162" shape="circle">
    <area target="" alt="CHINA" title="CHINA" href="china.html" coords="616,204,837,335" shape="rect">
</map>
</body>
</html>
```
